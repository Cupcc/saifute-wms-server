## 处理建议

这 `41` 行不能直接批量按单据价改库存流水。建议按下面顺序处理：

1. 下一批优先确认金额较大的 `033`、`yh1`。
2. 对每行回查同物料历史入库和旧系统原始单价，判断是“领料单价错”还是“缺少同价入库来源”。
3. 如果单据价正确但缺来源，应补齐或修复入库来源事实，再重跑批量修复。
4. 如果单据价错误，应修正车间单据明细成本口径，而不是强行把库存来源改成不存在的价格层。
5. 对 `0.12/0.10`、`0.89/0.88`、`2.67/2.65` 这类小价差行，应优先确认是否属于历史导入四舍五入或旧系统单价精度差异。
