## 处理建议

这批初始 `44` 行已全部闭环。后续同类问题仍按下面顺序处理：

1. 先回查同物料历史入库、旧系统原始单价和 `inventory_source_usage`，判断是“领料单价错”还是“缺少同价入库来源”。
2. 如果单据价正确但缺来源，应补齐或修复入库来源事实，再重跑批量修复。
3. 如果单据价错误，应修正车间单据明细成本口径，而不是强行把库存来源改成不存在的价格层。
4. 对 `1.05` 这类低金额行，应确认是否属于历史导入字段语义差异、旧系统单价精度差异，还是确实缺少同价来源。
